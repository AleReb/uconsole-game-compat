# Legal and redistribution boundaries

This repository contains original documentation and compatibility scripts only.

It does not distribute:

- game executables or installers;
- Unity assets, bundles, managed assemblies, textures, audio or saves;
- GOG/Steam runtime files;
- proprietary libraries;
- authentication material, cracks or DRM circumvention.

Binary patch scripts require the user to provide a legally obtained matching
file. They verify expected bytes/hashes, create a backup and modify the user's
local copy.

Do not attach generated backups, converted bundles, extracted textures or game
logs containing personal data to public issues.
